Проект по регрессии: Прогнозирование цен на дома в Эймсе, Айова
Введение
Этот проект основан на соревновании Kaggle "House Prices - Advanced Regression Techniques". Ваша задача - разработать модель машинного обучения для прогнозирования цен на жилые дома в Эймсе, штат Айова, используя продвинутые методы регрессии.
Источник данных
Набор данных доступен на Kaggle по следующей ссылке:
https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques
Описание данных
Набор данных содержит 79 объясняющих переменных, описывающих (почти) каждый аспект жилых домов в Эймсе, Айова. Ваша цель - предсказать конечную цену каждого дома.
Задачи проекта

Исследовательский анализ данных (EDA)

Проведите тщательный анализ данных, включая визуализацию распределений и корреляций.
Выявите наиболее важные факторы, влияющие на цену дома.
Исследуйте пропущенные значения и выбросы.

Разделение данных

Разделите данные на обучающую и тестовую выборки.


Построение базовой модели

Реализуйте простую линейную регрессию как базовую модель.
Оцените производительность базовой модели, используя метрики MAE, MSE и R².


Анализ важности признаков

Определите, какие характеристики бриллиантов наиболее важны для прогнозирования цены.


Визуализация результатов

Создайте графики, показывающие фактические vs предсказанные цены.
Визуализируйте остатки модели для проверки допущений.


Выводы и рекомендации

Обобщите ваши находки и предложите дальнейшие шаги для улучшения модели.
