Задание по регрессии: Прогнозирование цен на бриллианты
Цель задания
Разработать модель машинного обучения для прогнозирования цен на бриллианты на основе их характеристик.
Набор данных
Вы будете работать с классическим набором данных, содержащим информацию о почти 54,000 бриллиантах. Набор данных доступен на Kaggle по следующей ссылке:
https://www.kaggle.com/datasets/shivam2503/diamonds
Набор данных включает следующие характеристики:

price: цена в долларах США ($326--$18,823)
carat: вес бриллианта (0.2--5.01)
cut: качество огранки (Fair, Good, Very Good, Premium, Ideal)
color: цвет бриллианта, от J (худший) до D (лучший)
clarity: измерение прозрачности бриллианта (I1 (худший), SI2, SI1, VS2, VS1, VVS2, VVS1, IF (лучший))
x: длина в мм (0--10.74)
y: ширина в мм (0--58.9)
z: глубина в мм (0--31.8)
depth: общий процент глубины = z / mean(x, y) = 2 * z / (x + y) (43--79)
table: ширина верхней части бриллианта относительно самой широкой точки (43--95)

Задачи

Загрузка и исследование данных

Загрузите набор данных с Kaggle и выполните его первичный анализ.
Проведите разведочный анализ данных (EDA), включая визуализацию распределений и корреляций между признаками.




Разделение данных

Разделите данные на обучающую и тестовую выборки.


Построение базовой модели

Реализуйте простую линейную регрессию как базовую модель.
Оцените производительность базовой модели, используя метрики MAE, MSE и R².


Анализ важности признаков

Определите, какие характеристики бриллиантов наиболее важны для прогнозирования цены.


Визуализация результатов

Создайте графики, показывающие фактические vs предсказанные цены.
Визуализируйте остатки модели для проверки допущений.


Выводы и рекомендации

Обобщите ваши находки и предложите дальнейшие шаги для улучшения модели.


Формат сдачи
Размещение проекта на GitHub

Создайте новый репозиторий на GitHub для вашего проекта.
Инициализируйте Git в вашей локальной директории проекта, если еще не сделали это.
Добавьте все файлы проекта в репозиторий, включая:

Jupyter Notebook или Python-скрипт с полным анализом
README.md файл с описанием проекта, его целей и инструкциями по запуску
Файл requirements.txt с списком необходимых библиотек и их версий
Любые дополнительные файлы, использованные в проекте (например, файлы с данными, если это разрешено)

Удачи в выполнении задания!
